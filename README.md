# Results Summary Component - Frontend Mentor

Welcome to my solution for the Results Summary Component challenge! This project is a responsive score card built to display test results and performance summaries.

## Features

* **Responsive Design:** Adapts seamlessly to different screen sizes. Displays a side-by-side layout on Desktop and a stacked layout on Mobile.
* **CSS Gradients:** Utilized complex `linear-gradient` for the Result Section and Score Circle to match the design perfectly.
* **Flexbox:** Used CSS Flexbox for alignment, centering, and structure.
* **Semantic HTML:** Built with accessible tags like `<main>`, `<h1>`, and `<section>` for better SEO and accessibility.

## Technologies Used

* HTML5
* CSS3 (Variables & Flexbox)
* Git & GitHub
